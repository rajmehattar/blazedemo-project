Travel and Tourism Automation Project

1. Project Name:-
Automation Testing Framework for blazedemo.com flight booking website.

2. Objective:-
To design and implement a scalable automation framework to validate the core functionalities of blazedemo.com flight booking website using Selenium WebDriver and TestNG.

3. Technology Stack:-
•	Language: Java
•	Automation Tool: Selenium WebDriver
•	Test Framework: TestNG
•	Build Tool: Maven
•	Design Pattern: Page Object Model (POM)
•	Reporting: TestNG Reports and Extent Reports
•	Version Control: Git

4. Reporting:-
•	TestNG default reports
•	Extent Reports
